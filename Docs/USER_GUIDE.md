# User Guide — Lumina University Management System

## Вхід у систему

Відкрий браузер і перейди за адресою застосунку. На головній сторінці натисни **Login**.

| Роль          | Email                    | Пароль       |
|---------------|--------------------------|--------------|
| Адміністратор | admin@lumina.edu         | Admin123!    |
| Викладач      | teacher@lumina.edu       | Teacher123!  |
| Студент       | student@lumina.edu       | Student123!  |

Після входу ти потрапиш на Dashboard — головну панель з основною статистикою.



## Для адміністратора

### Управління студентами

**Перегляд списку:** меню → Students → список усіх студентів, відсортованих за прізвищем.

**Додавання студента:**
1. Students → Create
2. Заповни: ім'я, прізвище, email, дата народження, група, статус, рейтинг
3. Введи пароль для облікового запису студента
4. Натисни Save

Система автоматично створить обліковий запис Identity з роллю Student.

**Редагування:** у списку студентів натисни Edit навпроти потрібного студента.

**Видалення:** Delete → підтвердження → студент видалений.

**Деталі студента:** Details — повна інформація: оцінки по курсах, відвідуваність, розклад групи.


### Управління викладачами

Teachers → Create → заповни ім'я, прізвище, email, кафедру → Save.

Викладачу можна призначати курси через розділ Courses.


### Управління факультетами та групами

**Faculties → Create** — назва факультету, ПІБ декана.

**Groups → Create** — назва групи, факультет. До групи прив'язуються студенти та курси.


### Курси та заняття

**Courses → Create** — назва, кредити, опис, викладач, групи.

До курсу можна додавати заняття трьох типів:
- **Lecture** — лекція
- **Seminar** — семінар
- **Lab** — лабораторна робота


### Розклад

Schedule → Create — день тижня, час початку та кінця, група, заняття.

Студент бачить розклад своєї групи на сторінці Details свого профілю.


### Оцінки

Grades → Create — вибери студента, курс, введи оцінку (число), дату, коментар.


### Відвідуваність

Attendance → Create — студент, заняття, статус (Present / Absent / Late), дата.


### Звіти

Reports → Generate → вибери тип (PDF або Excel) → система генерує звіт та зберігає в архіві.

Reports → Index — перелік всіх згенерованих звітів з датою та автором.


### Користувачі

Users → список всіх зареєстрованих користувачів з ролями.

Адміністратор може змінювати ролі через Users → Edit.


## Для викладача

### Перегляд студентів

Students → список студентів. Клікни Details для перегляду повного профілю: оцінки, відвідуваність, розклад.

### Виставлення оцінок

Grades → Create → вибери студента та курс → введи оцінку.

### Відмітка відвідуваності

Attendance → Create → вибери студента та заняття → відміть статус.

### Сповіщення

Notifications → Send — надішли повідомлення студентам групи. Студенти отримають сповіщення в реальному часі без перезавантаження сторінки.


## Для студента

### Мій профіль

Students → Details (твій профіль) — бачиш:
- Особисті дані та рейтинг
- Оцінки по кожному курсу
- Статистику відвідуваності
- Розклад своєї групи

### Розклад

На сторінці профілю — блок розкладу з усіма заняттями групи, часом та курсом.

### Сповіщення

Notifications → список всіх отриманих сповіщень. Непрочитані виділені. Клікни на сповіщення щоб позначити як прочитане.

Нові сповіщення від викладачів з'являються автоматично без перезавантаження сторінки.


## Dashboard

Головна панель показує:
- Загальна кількість студентів, викладачів, курсів
- Середній рейтинг студентів
- Останні активності
- Швидкі посилання на основні розділи

## Часті питання

**Забув пароль?** Зверніться до адміністратора — він може скинути пароль через Users → Edit.

**Студент не відображається у списку групи?** Перевір, чи правильно вказано GroupId при створенні студента.

**Звіт не генерується?** Переконайся що є права адміністратора — Reports доступні тільки для Admin.
